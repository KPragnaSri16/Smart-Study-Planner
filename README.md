# 📘 Smart Study Planner

A responsive web-based study planning application designed to help students organize academic tasks, manage deadlines, track progress, and improve productivity through an interactive dashboard.

## 🚀 Features

* Add, edit, and delete study tasks
* Set task priorities (Low, Medium, High)
* Track task completion status
* Search and filter tasks efficiently
* Progress tracking with visual progress bar
* Timeline-based task visualization
* Calendar view for deadline management
* Local storage support for persistent task data
* Responsive and user-friendly interface

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* Browser Local Storage

## 📋 How It Works

1. Create a study task with a title, deadline, notes, and priority level.
2. Manage tasks using edit and delete functionalities.
3. Track completed and pending tasks through filters.
4. Monitor study progress using the built-in progress tracker.
5. Visualize upcoming deadlines through timeline and calendar views.
6. Data is automatically saved using browser localStorage.
